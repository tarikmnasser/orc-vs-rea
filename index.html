Histórico semanal
Um retrato é salvo automaticamente sempre que o painel é aberto e já se passou uma semana desde o último. Também dá pra forçar manualmente.
Backup dos dados
Os dados ficam só neste navegador/aparelho. Baixe um arquivo de backup de vez em quando (ex: toda sexta) e guarde no Drive/e-mail — se o navegador for limpo, você restaura a partir dele em qualquer aparelho.
⚠️ O mais importante
  • Use sempre o mesmo navegador e o mesmo aparelho. Os dados (tarefas concluídas, prazos, o que está aberto) ficam salvos só ali — abrir em outro navegador ou celular mostra tudo zerado.
  • Não limpe o cache/dados de navegação do navegador (nem "restaure o iPhone/Android de fábrica") sem antes baixar um backup — isso apaga tudo que está salvo aqui.
  • Baixe um backup com frequência (aba Resumo → "Baixar backup"). O ideal é toda sexta, ou sempre depois de uma atualização grande. Guarde o arquivo no Drive, e-mail ou WhatsApp "mensagens para mim".
  • Este painel não sincroniza entre pessoas. Se você e outra pessoa abrirem o mesmo link, cada um tem sua própria cópia local — marcar algo no seu aparelho não aparece no do outro.
Como usar — Mapa
  • Toque num nó para abrir/fechar os ramos dele e ver as ações (marcar concluído, renomear, prazo, adicionar, remover).
  • Arraste para mover o mapa; use os botões +/– ou pinça/scroll para dar zoom.
  • Um "+" do lado do número indica que aquele ramo tem itens escondidos — toque para abrir.
Como usar — Lista e Busca
  • Na Lista, o quadrado marca como concluído; a seta ▶ expande/recolhe.
  • A busca (no topo) mostra sugestões conforme você digita — clique numa sugestão, ou aperte Enter para ir direto no primeiro resultado.
  • O ✕ de remover sempre pede confirmação antes de excluir de verdade.
Dúvidas ou ajustes
Qualquer mudança no painel (novo campo, cor, estrutura) é feita direto no arquivo — fale com o Tarik.
